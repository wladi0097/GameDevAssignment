# Game Developer Assignment
## by Wladislaw Koschemako

### Web version

https://wladi0097.github.io/GameDevAssignment/


### Usage

Build:
````bash
npm run build
````

Watcher:
````bash
npm run watch
````

### Built with

* Jetbrains Webstorm
* Rollup as js bundler
* GitHub pages as web host, deployed via actions
* Eslint for dev time linting
* Typescript
* Pixi.js
